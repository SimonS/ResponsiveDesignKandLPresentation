Points to consider

  • Homepages are hard
  • Need to think about all layouts at the same time
  • Javascript behaviours that depend on layout are tricky
  • Media elements need careful consideration - video, images
  • TV views have not been included, yet

Our modular approach

  • Uses BBC Module (PAL based assembly an option)
  • Uses Cuts The Mustard (or similar technique) to enhance page on DomContentLoaded event
  • wraps module html, css and js in json
  • combines muliple modules in one request
  • highly cachable URLs (both base page and js loaded)

What we need/Outstanding Questions

  • better css management
  • can we do without jquery?
  • how do we monitor a page which is assembled in the browser?
  • video, images still not solved
